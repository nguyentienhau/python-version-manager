# Python Version Manager

## Basic Commands

    pvm version
    pvm current
    pvm commands
    pvm help
    pvm on
    pvm off
    pvm list [available]
    pvm global [version]
    pvm local [version]
    pvm install [version]
    pvm uninstall [version]

## Add Environment Variables

    PVM_HOME		C:\Users\Admin\Programs\Python Version Manager
    PVM_CURRENT		3.9.13

## Add Paths

    %PVM_HOME%
    %PVM_HOME%/versions/%PVM_CURRENT%
